Method used for SUBMIT_RESULTS/Bindi/bindi_brune_fit.csv:
- Brune model (n=2):  Mo/(1+(frequency/fc)^2)
- non-linear least-squares (nls function in R)
- frequency vector equally-spaced in log scale ('spec_log_',evID,'_shanna_test.csv' files are considered)
- fit in displacement over the frequency range 0.6-40 Hz

- header of the output csv file
 ID: event ID
 logMo: logarithm of the seismic moment (log 10) [Nm]
 logMo_95CI: value for computing 95% the confidence interval for LogMo 
 fc: corner frequency [Hz]
 fc_95CI: value for computing the 95% conficence interval for fc
 rse: residual standar error (computed as sqrt(sum(residuals^2)/degrees_of_freedom))
 cov_fcLogMo: off-diagonal element of the covariance matrix quantifing the correlation between LogMo and fc

where logMo_95CI is computed by multiplying the standard error on LogMo for the t-value corresponding to the available number of degrees of freedom and probability 0.975.
(e.g., error_LogMo*qt(0.975,deg_of_freedom)). The errors on LogMo and fc are extracted from the diagonal of the covariance matrix.

The seismic moments are recomputed but the values are constrained to be in agreement on average with catalog magnitudes above 3.5.
Figures of all fit are provided in the folder: SUBMIT_RESULTS/Bindi/FIGURE/FIGUREFITBRUNE
