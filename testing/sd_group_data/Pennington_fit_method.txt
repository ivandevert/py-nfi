Method:
Fitting the Brune model between 1-40 Hz using the log version of the data set. 
Code version:p(1)-log10(1+(t/p(2)).^2)
Minimization Method:
Matlab fminsearch function is being used to find the best fitting fc p(2). 
Starting fc point in inversion is set to 10. 
Omega (p(1)) in the fit is set to be mean amplitude of 1-3 Hz bandwidth range.  
Error estimate:
Is defined as the region of variance reduction within 5% of the minimum.
